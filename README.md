# 🌿 Green Grocer - Premium Organic Food Delivery

A clean, modern, and high-fidelity mobile application interface designed for an organic food delivery service. This project balances beautiful, natural aesthetics with absolute functional utility.

## 👤 About Me
Hi! I am a passionate UI/UX Designer and Frontend Developer. This project is a part of my industrial training as a **UI/UX Design Intern at DecodeLabs**, focusing on creating clean, user-centered digital experiences.

## 🚀 Key Features
* **The Happy Path Flow:** Optimized user journey across core screens (Homepage ➔ Search ➔ Add to Cart ➔ Checkout).
* **Thumb Zone Optimization:** Core buttons placed in the "Easy Reach" zone for comfortable one-handed mobile usage.
* **Micro-Interactions:** Component sets with smooth hover, pressed, and disabled states to build interactive system trust.
* **Native-Feel Carousel:** Swipeable banner built with Auto Layout and Smart Animate (Ease Out 300ms).
* **Guest Checkout:** Frictionless checkout option to increase conversion without mandatory signup walls.
